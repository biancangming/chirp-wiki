# chirp-wiki

> chirp 是一个小众好用的内网穿透工具，相比于frp、ngrock，作者贴心的适配了windows、macos、android 图形化客户端，linux 服务端和客户端拥有完善清晰的图形化管理页面。对我这种经常使用内网穿透的程序员来说，方便了许多。
> 通过我的体验，我觉得这款工具在体验上完全超越frp，主要是图形化的客户端操作起来太方便了，而且服务端的部署可以一键操作，不想frps，每次都让人感觉难受。
